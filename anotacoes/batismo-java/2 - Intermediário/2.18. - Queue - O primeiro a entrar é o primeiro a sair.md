A **Queue** em Java é uma estrutura de dados que segue o princípio FIFO (First In, First Out), onde o primeiro elemento inserido é o primeiro a ser removido. 

Ela faz parte do pacote `java.util` e é amplamente utilizada quando a ordem de processamento dos elementos é importante, como em filas de espera.


## **FIFO**
Em uma fila, o primeiro elemento adicionado é o primeiro a ser removido, ao contrário de outras estruturas como pilhas (`Stack`), que seguem o princípio LIFO (Last In, First Out).
## **Implementações**
As implementações mais comuns de `Queue` em Java incluem `LinkedList`, `PriorityQueue`, e `ArrayDeque`. Cada uma tem características e usos específicos, dependendo do tipo de operação que você precisa realizar.
## **Operações**
A interface `Queue` define métodos importantes como `add()`, `offer()`, `poll()`, `peek()`, entre outros. O método `add()` insere um elemento na fila, e o método `poll()` remove e retorna o primeiro elemento.
## **Exemplo no Contexto de Naruto**
Imagine que você tem uma fila de missões para os ninjas completarem, onde cada missão precisa ser tratada na ordem em que foi recebida. A `Queue` pode ser usada para armazenar e processar essas missões, garantindo que a missão mais antiga seja completada primeiro.

## Mais exemplos

```Java
Queue<String> ninjasQueue = new LinkedList<>();  
ninjasQueue.offer("Naruto");  
ninjasQueue.offer("Sasuke");  
ninjasQueue.offer("Hinata");  
ninjasQueue.offer("Sakura");  
ninjasQueue.add("Konha");  
ninjasQueue.poll();  // Retira o Head
System.out.println(ninjasQueue); // Hinata, Sakura, Konha
```
