Uma **Classe Utilitária** é um contêiner para um grupo de métodos estáticos relacionados que executam operações comuns e reutilizáveis.

Sua principal característica é que essas operações são **stateless**, não dependem do estado de nenhuma instância de objeto. Elas operam exclusivamente com base no argumentos que recebem.

Pense em **Classes Utilitárias** como uma caixa de ferramentas para uma finalidade específica. Você não precisa de uma *instância* de caixa de ferramentas para usar um martelo, você simplesmente pega o martelo e o utiliza.

Você não instância uma classe utilitária, você só invoca seus métodos estáticos diretamente.

```Java
Collections.sort();
Arrays.asList();
Math.max();
```
